This example shows how to work with CSV data.

The etl.xml in csv directory use CSV driver to work with CSV files.
Type scriptella or ant to run the example.

The etl.xml file in the text directory use Text driver to work with CSV files.
Please note that Text driver is a generic Regex based tool which does not cover
escaping of CSV data.

The hsqldb directory contains ETL file to work with HSQLDB text tables.
