Janino Driver for Scriptella.

This driver allows to embed Java code into <query> and <script> elements. This code is compiled by Janino and executed by Scriptella engine.

Janino scripting elements are implicit subclasses of Query and Script base classes.

General information

Driver class:scriptella.spi.janino.Driver
Runtime dependencies:janino.jar

Driver Specific Properties

Name Description Required

Example

    <connection driver="janino">/>

    <query>
        set("name", "John);
        next();
        <script>
            System.out.println("Processing: "+get("name"));
        </script>
    </query>
