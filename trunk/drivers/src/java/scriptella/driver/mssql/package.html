Microsoft SQL Server Driver Adapter for Scriptella.

Note: The driver tries to load Microsoft SQL Server JDBC driver.

General information

Driver class:scriptella.driver.mssql.Driver
URL:Same as for Microsoft SQL Server JDBC Driver.
Runtime dependencies: sqljdbc.jar mssqlserver.jar mssql.jar

Driver Specific Properties

Name Description Required

Examples

    <connection id="c1" driver="mssql" url="jdbc:sqlserver://localhost;database=test" user="username" password="password">
    </connection>
    <connection id="c2" driver="mssql" url="jdbc:jtds:sqlserver://pc613/test" user="username" password="password">
    </connection>
