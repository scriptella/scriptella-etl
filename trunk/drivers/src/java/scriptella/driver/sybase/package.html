Sybase Driver Adapter for Scriptella.

Note: The driver tries to load one after another jdbc3,jdbc2 and jdbc Sybase drivers.

General information

Driver class:scriptella.spi.sybase.Driver
URL:Same as for Sybase JDBC Driver.
Runtime dependencies: jconn3.jar

Driver Specific Properties

Name Description Required

Examples

    <connection driver="sybase" url="jdbc:sybase:Tds:localhost:2638/tst" user="username" password="password">
    </connection>
