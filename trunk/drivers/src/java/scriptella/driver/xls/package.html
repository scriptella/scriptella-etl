DB2 Driver Adapter for Scriptella.

General information

Driver class:scriptella.driver.xls.Driver
URL:Same as for SQLSheet JDBC Driver.
Runtime dependencies:
  • sqlsheet.jar
  • poi.jar
  • jsqlparser.jar

Driver Specific Properties

Name Description Required

Examples

    <id="xls" url="jdbc:xls:file:report.xls" classpath="sqlsheet-0.1.0.jar; poi-3.0.2-FINAL-20080204.jar; jsqlparser.jar">
    </connection>

    <script connection-id="xls">
        CREATE TABLE SHEET1(
		    COL1 INT,
		    COL2 INT
	    );
    </script>
    ...
    <query connection-id="db">
        ...
        <script connection-id="xls">
            INSERT INTO SHEET1 (COL1,COL2) VALUES(${rownum},${col2_value});
        </script>
    </query>
