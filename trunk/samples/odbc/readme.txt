This example demonstrates working with ODBC datasources.

Velocity is used to produce the report based on a SQL query.
Janino is used to write images for HTML report.

Usage
-----
IMPORTANT: In order to run this example obtain Northwind.mdb - the
MS Access sample database. Put Northwind.mdb to the current directory.

Just type ant (for running from ant build file) or scriptella (to invoke
without Ant bridge).
